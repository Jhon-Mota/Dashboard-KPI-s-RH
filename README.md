# Dashboard-KPI-s-RH

Dashboard desenvolvido no Power BI para monitoramento dos principais indicadores de Recursos Humanos, com filtro por departamento e visão consolidada da empresa.

## Indicadores

O relatório apresenta total de banco de horas, taxa de faltas, taxa de atrasos, distribuição por gênero e custos de vale refeição — todos segmentáveis por departamento.

## Tecnologias

- Power BI Desktop;
- DAX para métricas calculadas;
- Arquivo CSV como fonte de dados.
